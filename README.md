# Personal E-Portfolio - Yash Pratap Singh

A modern portfolio website built with Next.js and React to showcase technical skills, education, projects, and achievements.

## Features

- Responsive single-page portfolio layout
- Animated hero section and custom interactions
- Technical skills dashboard with accessible controls
- Project, education, and certification sections
- SEO metadata, sitemap, robots, and Open Graph image support

## Portfolio Requirements Checklist

Before publishing this portfolio, update these personal assets and links:

- Add your GitHub profile URL.
- Add the live deployment URL and set `NEXT_PUBLIC_SITE_URL`.
- Add GitHub repository links for each featured project.
- Add live demo links for projects if available.
- Add a real resume PDF if you want a download button instead of the current email request CTA.
- Add certificate verification links, issuers, and completion dates.
- Add project screenshots or case-study images.
- Add measurable project outcomes such as performance improvements, user impact, features delivered, or publication links.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This portfolio is ready for deployment on platforms like Vercel. Set `NEXT_PUBLIC_SITE_URL` to your production domain before publishing so metadata, sitemap, and robots point to the correct URL.
